# 项目说明
于时同行团队打造-时光邮局项目
# 项目设计
- 采用VitePress+Vue3实现基本技术架构
- UI框架PC端页面采用ElementPlus作为基础UI框架
- 移动端采用Vant作为基础UI框架
- 全局使用TypeScript
- 使用Axios作为请求库（二次封装请求器与拦截器）
- 使用qs模块解包
- 使用Vditor作为Markdown编辑器（自定义初始化）
# 项目后端
> 后端SpringBoot+Redis+Quartz+Mp

主要功能：
- Redis实现API接口防刷节流
- Quartz实现定时任务调度
- 使用Mp进行持久化

> ** 详细请见后端仓库 **
